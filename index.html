NCL Eclipse: Editor Avançado de XML-NCL desenvolvido como um plug-in para a IDE Eclipse. O NCL Eclipse ajuda no desenvolvimento de aplicações hipermídia interativas usando a linguagem declarativa padrão para o desenvolvimento de aplicações para o Sistema Brasileiro de TV Digital Terrestre, a NCL. Possui recursos como marcação de erros, visão de esboço de contextos e autocompletação. Para mais informações, visite o NCL Eclipse web site.

Concluídos:

GT-VOA, 2015 - 2016: O projeto GT-VoA visa permitir a criação e exibição de objetos de aprendizagem (OAs), que são utilizados para o ensino à distância (EaD) como conteúdo multimídia interativo, e armazenados e distribuídos utilizando serviços do repositório de vídeos do serviço de vídeo sob demanda da RNP (Vídeo@RNP). Durante a primeira fase do projeto, a ferramenta de autoria Cacuriá foi implementada. O foco da ferramenta Cacuriá é permitir que profissionais da área de educação possam criar objetos de aprendizagem através de um processo pouco complexo. O serviço iVoD também foi especificado, de forma a permitir o armazenamento e distribuição desses objetos de aprendizagem integrados ao serviço Vídeo@RNP.

Sagga - Suporte à Geração Automática de Aplicações para o GINGA-NCL, 2015 - 2016: Este projeto tem como objetivo a geração automática de aplicações NCL 3.0. Como o Ginga-NCL é o padrão brasileiro de TV digital terrestre [ABNT NBR 15606-2 e ABNT NBR 15606-5] e padrão ITU-T para serviços IPTV [ITU-T H.761], aplicações podem ser implementadas tanto no SBTVD, como para redes IP, em particular a Internet. Para isso, as implementações do Ginga-NCL em código aberto, tanto para Linux e Windows, tanto para SBTVD como IP cliente/servidor podem ser usadas, além de implementações comerciais do Ginga.
