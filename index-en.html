Participant in the following projects:

Current:

NCL Eclipse: XML-NCL advanced editor designed as a plug-in for the Eclipse IDE. The NCL Eclipse supports the development of interactive hypermedia programs using the standard declarative language for developing programs for the Brazilian Terrestrial Digital TV, NCL. It has features like dialing errors, contextual outline view and autocomplete. For more information, see NCL Eclipse web site.

Concluded:

GT-VOA, 2015 - 2016: The GT-VoA project aims to enable the creation and display of learning objects (LOs), which are used for distance learning (DL) as interactive multimedia content, and stored and distributed using services repository of video service video on demand RNP (Video @ RNP). During the first phase of the project, Cacuriá authoring tool was implemented. The focus of Cacuriá tool is to allow education professionals can create learning objects through a process somewhat complex. The IVOD service was also specified to allow the storage and distribution of integrated learning objects to the service Video @ RNP.

SAGGA - Support to automatic generation of application for GINGA-NCL, 2011 - 2012: This project aims to the automatic generation of NCL 3.0 applications. As Ginga-NCL is the Brazilian standard for digital terrestrial TV [ABNT NBR 15606-2 and ABNT NBR 15606-5] and ITU-T standard for IPTV [ITU-T H.761] services, applications may be implemented both in SBTVD, as for IP networks, particularly the Internet. For this, the Ginga-NCL implementations in open source, for both Linux and Windows, both for SBTVD as Client/Server IP can be used, besides obviously commercial implementations Ginga.
